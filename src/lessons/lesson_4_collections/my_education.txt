Java Collections Framework (JCF) — це набір класів та інтерфейсів, які забезпечують структури даних та алгоритми для роботи з колекціями об'єктів. Колекції допомагають зберігати та обробляти групи даних з максимальною ефективністю. Нижче наводяться основні аспекти JCF, які варто знати при вивченні цієї теми.

                    ===============================
                    ==  Основні інтерфейси JCF  ===
                    ===============================

                            Collection

Базовий інтерфейс для більшості колекцій.
Методи: add(), remove(), size(), clear(), isEmpty(), contains(), iterator(), та інші.


                    List
Розширює Collection. Зберігає елементи у впорядкованому списку.
Реалізації: ArrayList, LinkedList, Vector, Stack.

                    Set
Розширює Collection. Зберігає унікальні елементи (жодних дублікатів).
Реалізації: HashSet, LinkedHashSet, TreeSet.

                    Queue
Розширює Collection. Зберігає елементи у черзі.
Реалізації: LinkedList, PriorityQueue, ArrayDeque.

                    Map
Зберігає пари ключ-значення. Не є прямим підкласом Collection.
Реалізації: HashMap, LinkedHashMap, TreeMap, Hashtable, Properties.



                    ==========================
                    ==  Основні класи JCF  ===
                    ==========================
                            ArrayList
Динамічний масив, який дозволяє швидкий доступ до елементів за індексом.
Повільний при вставці та видаленні елементів з середини списку.

                            LinkedList
Двоспрямований зв'язний список, що забезпечує швидку вставку та видалення.
Повільний при доступі до елементів за індексом.

                            HashSet
Використовує хеш-таблицю для зберігання унікальних елементів.
Забезпечує швидкий доступ, вставку та видалення.

                            HashMap
Зберігає пари ключ-значення у хеш-таблиці.
Забезпечує швидкий доступ до значень за ключами.


                    ===========================
                    ==   Основні методи JCF  ==
                    ===========================

   - add(E e): додає елемент до колекції.
   - remove(Object o): видаляє зазначений елемент з колекції.
   - size(): повертає кількість елементів у колекції.
   - clear(): видаляє всі елементи з колекції.
   - isEmpty(): повертає true, якщо колекція порожня.
   - contains(Object o): перевіряє, чи міститься вказаний елемент у колекції.
   - iterator(): повертає ітератор для обходу елементів колекції.

Висновок
Java Collections Framework надає потужний набір інструментів для роботи з колекціями даних. Знання основних інтерфейсів і класів JCF допоможе ефективно зберігати, обробляти і маніпулювати даними в Java програмах.
